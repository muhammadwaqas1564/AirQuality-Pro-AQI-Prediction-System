About AQI

The Air Quality Index (AQI) is used by government agencies to communicate to the public how polluted the air currently is or how polluted it is forecast to become.

Health Implications

AQI values help determine potential health effects of air pollution

Precautionary Actions

Based on AQI levels, precautions can be taken to minimize exposure

AQI Scale

0-50 51-100 101-150 151-200 201+
Good Moderate Unhealthy (Sensitive) Unhealthy Very Unhealthy

How It Works

Our system uses machine learning models to predict AQI values based on input parameters for various pollutants.

  • Decision Tree: Non-linear model good for classification
  • SVR: Support Vector Regression for continuous values
  • KMeans: Clustering algorithm for pattern detection
